### 爱上阅读

#### demo 演示 https://likeread.herokuapp.com/

#### 启动项目

- 1、git clone
- 2、cd likeread
- 3、npm install
- 4、npm start

#### nodejs 功能开发

- nodeJs express Router 路由分发；
- nodeJs 使用了的模块，fs ，path；
- 开发书籍的增删改查；
- 数据操作使用 books.json 存储；
- 页面 ejs 模板引擎，使用主模板 layout.html 文件，其他有 index（首页）, addBook（添加书籍）, login（登录）, modifyBook（修改书籍）, success（成功跳转）；
